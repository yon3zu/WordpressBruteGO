## _features_

- Auto Get Username
- 4 Type BruteForce Combination
  - {USERNAME}+{PASSWORD)
  - {USERNAME}+{USERNAME}{PASSWORD)
  - {USERNAME}+{PASSWORD){USERNAME}
  - {USERNAME}+{USERNAME}{USERNAME}
- Multithreading
- Fast Tools
- Build In GOLANG Latest Version (20.5)
